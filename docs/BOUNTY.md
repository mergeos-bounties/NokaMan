# NokaMan bounties (MergeOS MRG)

NokaMan pays contributors in **MRG** through the MergeOS bounty program.

## Before you start

1. **Follow** the org https://github.com/mergeos-bounties  
2. **Star** https://github.com/mergeos-bounties/mergeos  
3. **Star** https://github.com/mergeos-bounties/mergeos-contracts
4. Comment on the NokaMan issue: `I claim this bounty`  
5. Comment on MergeOS [Claim Token #1](https://github.com/mergeos-bounties/mergeos/issues/1) with the issue link  
6. Open a PR to **NokaMan** `master` (`Fixes #N`)

## Payout scale

| Scope | Typical MRG |
| --- | ---: |
| Small / good-first | 25 |
| Medium feature | 50 |
| Large system piece | 100 |
| Extra-large | 200 |

Issue titles may show marketing amounts; **admin ledger pays 25/50/100/200** after review.

## Domains we fund

- Multi-language rubrics (EN/KO/JA/VI/ZH/…)  
- Writing / speaking / listening assessors  
- CEFR / JLPT / TOPIK / TOEIC-style mapping  
- Calibration datasets & metrics  
- LLM-backed graders (optional extras)  
- FastAPI / SDK for language-learning apps  
- Accessibility & fairness docs  

## Evidence

- Tests + command output for backend/ML  
- Screenshots for UI demos  
- Dataset license notes for any new data
